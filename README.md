# pupismyname.com

https://www.pupismyname.com/

Static site generated by 11ty


## Install
Requires Node 10+ (for 11ty)
```
npm ci
```


## Build
```
npm run build
```
builds to `_site`


## Dev
```
npm run dev
```
spins up http://localhost:8169 and watches for changes



## Clean
```
npm run clean
```
removes 11ty build folder, automatically run before `dev` and `build`
